Default

$('#target').blinky();

With fixed value of columns

$('#target').blinky({ numberOfColumns: 5 });

Disable blinking (color changing)

$('#target').blinky({ blinking: false });

Resizing every second

$('#target').blinky();
setInterval(function() {
    $('#target').width(chance.natural({min: 150, max: 600}));
}, 1000);
