# Blinky

useless plugin for [jQuery][1] with blinking colors and all


## Requirements

* [jQuery][1]
* [Chance][2]


## Usage

    $('#element').blinky();

or

    $('#element').blinky({
        option1: 'value'
        option2: 'value'
    });


## Options

### numberOfColums

* type: `integer`
* default: random value between `5` and `50`

### blinking

controls the color changing

* type: `boolean`
* default: `true`
